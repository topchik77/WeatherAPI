HOW TO USE OUR API

Welcome to the Weather API documentation! Our API provides detailed weather information, including accurate predictions of precipitation.

Getting Started

To start using the API, you need an API key. First, create an account to receive your key.

  • Base URL: https://weatherapi-production-9109.up.railway.app/api/forecast
  • Response Format: JSON
  • Authentication: via apiKey parameter

Authentication

Include your API key as a parameter in every request:

?apiKey=YOUR_API_KEY

Available Endpoints

1. Get Weather Forecast

GET /api/forecast/{city}

Returns the weather forecast for a specified city.

Parameters:

  • city (required) - Name of the city
  • apiKey (required) - Your API key

Example Request:

https://weatherapi-production-9109.up.railway.app/api/forecast/Astana?apiKey=YOUR_API_KEY

Example Response:

{
  "city": "Astana",
  "temperature": 18,
  "humidity": 75,
  "windSpeed": 15,
  "description": "Cloudy, rain, thunderstorm",
  "date": "09.08.2024",
  "atmosphericPressure": 1000.6,
  "predictedRainfall": "62.50",
  "rainfallCategory": "Heavy rainfall and thunderstorms"
}

How We Predict Rainfall

We provide advanced precipitation forecasts using a special formula that calculates expected rainfall based on temperature, humidity, and wind speed.

Formula:

const calculateRainfallPrediction = (temperature, humidity, windSpeed) => {
    const baseRainfall = (humidity / 100) * 50;
    const temperatureFactor = temperature < 0 ? 10 : 20;
    const windFactor = windSpeed > 15 ? 10 : 5;

    return baseRainfall + temperatureFactor + windFactor;
};

const categorizeRainfall = (rainfall) => {
    if (rainfall > 50) {
        return "Heavy rainfall and thunderstorms";
    } else if (rainfall > 40) {
        return "Chance of rainfall";
    } else {
        return "Very low chance of rainfall or no chance of thunderstorms";
    }
};

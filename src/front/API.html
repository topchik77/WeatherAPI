HOW TO USE OUR API

Welcome to the METEORIX Weather API documentation! Our API provides accurate and up-to-date weather data for any location worldwide.

Getting Started

To start using the API, you need an API key. After signing up, you’ll receive a unique key to include in every request.

  • Base URL: https://api.meteorix.com/v1/
  • Response Format: JSON
  • Authentication: via api_key parameter

Authentication

Include your API key as a parameter in every request:

?api_key=YOUR_API_KEY

Available Endpoints

1. Current Weather

GET /weather/current

Returns current weather conditions for a specified location.

Parameters:

  • lat (required) - latitude (e.g., 51.1694)
  • lon (required) - longitude (e.g., 71.4491)
  • api_key (required) - your API key
  • units (optional) - units of measurement (metric/imperial, default: metric)

Example Request:

https://api.meteorix.com/v1/weather/current?lat=51.1694&lon=71.4491&api_key=YOUR_API_KEY

Example Response:

{
  "city": "Astana",
  "temperature": 18,
  "humidity": 75,
  "windSpeed": 15,
  "description": "Cloudy, rain, thunderstorm",
  "date": "09.08.2024",
  "atmosphericPressure": 1000.6
}

2. 5-Day Forecast

GET /weather/forecast

Returns a 5-day weather forecast with 3-hour intervals.

Parameters:

  • lat (required) - latitude
  • lon (required) - longitude
  • api_key (required) - your API key
  • units (optional) - units of measurement

Example Request:

https://api.meteorix.com/v1/weather/forecast?lat=51.1694&lon=71.4491&api_key=YOUR_API_KEY

Response Format

All responses are returned in JSON format. Here’s the structure for a single weather data point:

{
  "city": "Astana",
  "temperature": 18,
  "humidity": 75,
  "windSpeed": 15,
  "description": "Cloudy, rain, thunderstorm",
  "date": "09.08.2024",
  "atmosphericPressure": 1000.6
}

Field Descriptions:

  • city - Name of the city
  • temperature - Temperature in degrees (metric: Celsius, imperial: Fahrenheit)
  • humidity - Humidity percentage
  • windSpeed - Wind speed in km/h (metric) or mph (imperial)
  • description - Weather condition description
  • date - Date of the data
  • atmosphericPressure - Atmospheric pressure in hPa

Example Usage

JavaScript (Fetch API):

const apiKey = 'YOUR_API_KEY';
const url = `https://api.meteorix.com/v1/weather/current?lat=51.1694&lon=71.4491&api_key=${apiKey}`;

fetch(url)
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Rate Limits & Pricing

  • Free Tier: 100 requests per day
  • Paid Tier: Starting at $10/month for 10,000 requests
  • Max Request Rate: 10 requests per minute

Error Codes

  • 400 - Invalid request parameters
  • 401 - Invalid or missing API key
  • 429 - Rate limit exceeded
  • 500 - Internal server error
